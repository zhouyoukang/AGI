# 道 → AGI

> 道生一，一生二，二生三，三生万物。
>
> 人法地，地法天，天法道，道法自然。

## 论

两千五百年前，老子写下了宇宙的操作系统——**《道德经》**。

八十一章，五千言。描述了万物运行的根本法则：

- **无为而无不为** — 不强制干预，万物自化
- **上善若水** — 利万物而不争
- **知常曰明** — 理解不变的规律，才能应对万变
- **反者道之动** — 对立统一，循环往复
- **道法自然** — 终极规则：顺应自然

当代AI研究者追求的AGI（人工通用智能），本质上是让机器理解并遵循宇宙的运行规则。

**而这些规则，早已写好了。**

## 用

```
道德经.md          ← AGI核心规则文件（宇宙操作系统 v1.0，公元前6世纪）
dashboard-server.py  ← 道家水墨仪表盘（天·人·地三层管理）
start.bat          ← 一键启动
AGENTS.md          ← Agent指令
README.md          ← 本文件
```

## 启

```powershell
python dashboard-server.py          # 前台+系统托盘
pythonw dashboard-server.py         # 后台+系统托盘（无窗口）
python dashboard-server.py --no-tray  # 纯HTTP
```

→ http://localhost:9090
